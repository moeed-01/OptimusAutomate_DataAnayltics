🍽️ Task 1: Exploratory Data Analysis (EDA)

File: Task1_EDA_Tips.ipynb

What I did: I played data detective on a Restaurant Tips dataset to uncover what truly drives their daily revenue and customer tipping behavior.

How I did it: I used pandas to clean missing data, cap extreme outliers, and encode categories. Then, I generated 8 statistical visualizations using seaborn and matplotlib to extract actionable business recommendations.

📈 Task 2: Sales Performance Dashboard

File: Task2_Sales_Dashboard.ipynb

What I did: I built a fully interactive Superstore sales dashboard that instantly recalculates KPIs (Revenue, Orders) and redraws charts based on user-selected filters.

How I did it: I went beyond static charts by combining ipywidgets (for interactive date-pickers and dropdowns) with plotly (for live charts) to create a dynamic web-app experience right inside a single Colab notebook.

🌍 Task 3: COVID-19 / Global Trends Analysis

File: Task3_Covid_Trends.ipynb

What I did: I analyzed the historical time-series footprint of COVID-19, comparing the explosive variant spikes in the US and UK against the much more stable growth pattern observed in Pakistan.

How I did it: I wrote a script to pull live data directly from the Johns Hopkins University GitHub repo, reshaped the matrix using pandas, and plotted time-series curves to visually identify major anomalies over a 3-year period.

Environment: Google Colab(python 3)

Data Manipulation: pandas, numpy

Visualization: matplotlib, seaborn, plotly

Interactive UI: ipywidgets
